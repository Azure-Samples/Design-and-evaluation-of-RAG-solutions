What is DNS?

DNS stands for Domain Name System. It is akin to an address book for the internet, where domain names like "www.quickconnect.com" are translated into machine-friendly IP addresses like "192.0.2.1". Without DNS, users would need to remember complex numerical addresses to navigate the web.

Term Description
Domain Name A human-readable address for websites (e.g., www.example.com).
IP Address A numerical label assigned to each device connected to a computer network (e.g., 192.0.2.1)
DNS Server A server that contains a database of public IP addresses and their associated hostnames.
DNS Query A request made by a user's computer to a DNS server to resolve a domain name.

Why Change DNS Settings?

There are several reasons why a customer might need to change their DNS settings:

  • Faster Internet Speeds: Some DNS servers are faster than others, which can improve website load times.
  • Better Security: Certain DNS providers offer enhanced security features, such as phishing protection.
  • Content Filtering: Some DNS services provide parental controls or block access to harmful websites.
  • Access to Restricted Content: Changing DNS can help bypass geographical restrictions on some websites.

Step-by-Step Guide to Changing DNS Settings

This section provides a detailed guide for changing DNS settings on various devices and operating systems.

Device/OS Steps
Windows 10
  1. Open the Control Panel.
  2. Navigate to Network and Sharing Center.
  3. Click on Change adapter settings.
  4. Right-click on the network you are connected to and select Properties.
  5. Select Internet Protocol Version 4 (TCP/IPv4) and click Properties.
  6. Choose Use the following DNS server addresses.
  7. Enter the preferred and alternate DNS server addresses.
  8. Click OK to save the changes.
MacOS
  1. Open System Preferences.
  2. Click on Network.
  3. Select the network you are connected to and click Advanced.
  4. Navigate to the DNS tab.
  5. Click the + button to add new DNS server addresses.
  6. Enter the preferred and alternate DNS server addresses.
  7. Click OK to save the changes.
  8. Click Apply.
Android
  1. Open the Settings app.
  2. Tap on Network & Internet.
  3. Select Wi-Fi.
  4. Tap on the network you are connected to and hold it down until you see the Modify network option.
  5. Select Advanced options.
  6. Change IP settings to Static.
  7. Enter the preferred and alternate DNS server addresses.
  8. Tap Save.
iOS
  1. Open the Settings app.
  2. Tap on Wi-Fi.
  3. Tap on the network you are connected to.
  4. Scroll down and tap on Configure DNS.
  5. Select Manual.
  6. Remove existing DNS servers by tapping the - button.
  7. Tap Add Server to add new DNS server addresses.
  8. Enter the preferred and alternate DNS server addresses.
  9. Tap Save.
Router
  1. Access your router's settings by entering its IP address in a web browser.
  2. Login with your username and password (refer to router documentation).
  3. Navigate to the WAN or Internet settings.
  4. Locate the DNS Settings.
  5. Enter the preferred and alternate DNS server addresses.
  6. Save the changes and reboot your router if necessary.
